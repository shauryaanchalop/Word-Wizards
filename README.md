The official landing page for **Word Wizards**, the premier First-Year Student Development Society at Dr. Akhilesh Das Gupta Institute of Technology & Management (ADGIPS). 

This project is a fully responsive, cinematic, single-page application (SPA) built to showcase the society's legacy, departments, events, and leadership council with a highly premium, modern aesthetic.

---

## ✨ Key Features

* **🎥 Cinematic Video Hero:** A top-anchored, highly compressed background video with an elegant dark-gradient overlay and floating ambient lighting.
* **📱 Responsive Glassmorphism UI:** A sleek, borderless "floating pill" navigation bar that reacts to scroll events and adapts to a frosted mobile hamburger menu.
* **🎭 3D Interactive Council Cards:** Advanced CSS perspective animations that react to mouse movement. Clicking a card triggers a smooth, blurred modal revealing the member's details and direct social media links (Instagram/LinkedIn).
* **⏳ Dynamic Legacy Tabs:** A custom-built JavaScript tab system that seamlessly switches between the current Council 3.0 and historical data (Council 2.0 & Council 1.0) without reloading the page.
* **✨ Micro-Interactions:** Includes an infinite auto-scrolling marquee, a custom magnetic cursor for desktop users, and elegant scroll-reveal animations.

---

## 🛠️ Tech Stack

This project was built from scratch without heavy frameworks to ensure maximum performance and total control over the UI/UX.

* **HTML5:** Semantic structure and SPA sectioning.
* **CSS3:** Advanced styling including `backdrop-filter`, 3D transforms, CSS variables, CSS grid/flexbox, and keyframe animations.
* **Vanilla JavaScript:** DOM manipulation, custom cursor logic, modal handling, tab-switching, and scroll-event listeners.
* **Fonts:** *Cinzel Decorative* (Headers), *Playfair Display* (Subheaders), and *Poppins* (Body) via Google Fonts.
